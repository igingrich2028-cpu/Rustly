## 🚀 Features

### 🦀 Rust in the Browser

Write and execute Rust code instantly without installing `rustc` or `cargo`.

---

### 📦 40+ Popular Crates Supported

Rustly includes built-in support for **40+ popular Rust crates**, allowing you to use commonly needed libraries without manual setup.

Examples include:

- `serde`
- `tokio`
- `reqwest`
- and many more

---

### 📁 Full File System Support

Manage your project like a real Rust workspace:

- Create new files  
- Edit existing files  
- Delete files  
- Organize project structure  

---

### ⚙️ `config.toml` Support

Rustly supports project configuration via a `config.toml` file, allowing you to:

- Configure dependencies  
- Adjust project settings  
- Customize build behavior  

---

### 🖥 Integrated Terminal

See compiler output, errors, and program results in an embedded terminal panel.

---

### ⚡ Zero Setup

Everything runs in the browser — no downloads, no toolchains, no configuration required.
